## Списки

Чтобы добавить ненумерованные списки необходимо перед текстом поставить символ *. Например:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки необходимо перед текстом поставить цифру порядкого номера и точку. Например:
1. Первый элемент
2. Второй элемент
3. Третий элемент
